# Simple Android App

A basic Android application built with Java.

## Project Overview

This is a simple Android app that demonstrates a basic activity with a standard layout.

## Requirements

- Android Studio
- JDK 8 or higher
- Android SDK (API 24 minimum, API 34 target)

## Build Instructions

1. Open the project in Android Studio
2. Sync Gradle files
3. Build the project using `Build > Make Project` or run `./gradlew build`
4. Run the app on an emulator or connected device

## Project Structure

- `app/src/main/java/com/example/simpleapp/` - Main application code
- `app/src/main/res/` - Resources (layouts, strings, colors)
- `app/build.gradle` - App-level build configuration

## Dependencies

- AndroidX AppCompat
- Material Design Components
- ConstraintLayout

## Minimum SDK

- Minimum SDK: 24 (Android 7.0)
- Target SDK: 34 (Android 14)
